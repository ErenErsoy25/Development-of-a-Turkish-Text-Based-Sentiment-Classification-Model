# Text Emotion Detection Application

This project is a web application for detecting emotions in texts. The goal is for users to write texts expressing different emotions and get scores based on the emotion classification of these texts.

## Requirements

The following libraries need to be installed for the project to run:

- streamlit
- pandas
- numpy
- altair
- translate
- joblib
- scikit-learn

## Installation

You can install the required libraries using the following command:

```sh
pip install streamlit, pandas ,numpy ,altair ,translate ,joblib ,scikit-learn


After running the code for the first time, you need to type "streamlit run app.py" in the terminal, and then you will be redirected to the website.
